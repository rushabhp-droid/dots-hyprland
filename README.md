# ⌨️ Hyprland Keybindings & Gestures

This document outlines the keyboard shortcuts, mouse bindings, and trackpad gestures configured in this Hyprland setup.

> **Legend:**  
> `SUPER` = Windows / Command Key  
> `LMB` = Left Mouse Button  
> `RMB` = Right Mouse Button  

---

## 🚀 Applications

| Keybinding | Action |
| :--- | :--- |
| `SUPER` + `Q` | Open Terminal (`alacritty`) |
| `SUPER` + `E` | Open File Manager (`dolphin`) |
| `SUPER` + `R` | Open App Launcher (`rofi`) |

## 🪟 Window Management

| Keybinding | Action |
| :--- | :--- |
| `SUPER` + `C` | Close active window |
| `SUPER` + `V` | Toggle floating mode |
| `SUPER` + `F` | Toggle fullscreen |
| `SUPER` + `P` | Toggle pseudo-tiling |
| `SUPER` + `J` | Toggle split (dwindle layout) |
| `SUPER` + `M` | System shutdown / Exit Hyprland |

## 🧭 Focus & Workspaces

| Keybinding | Action |
| :--- | :--- |
| `SUPER` + `↑ / ↓ / ← / →` | Move focus in direction |
| `SUPER` + `[1-0]` | Switch to workspace 1-10 |
| `SUPER` + `Scroll Up/Down`| Scroll through adjacent workspaces |
| `SUPER` + `S` | Toggle special workspace (scratchpad) |

## 📦 Moving Windows

| Keybinding | Action |
| :--- | :--- |
| `SUPER` + `SHIFT` + `[1-0]` | Move active window to workspace 1-10 |
| `SUPER` + `SHIFT` + `S` | Move active window to special workspace |

## 📸 Utilities & Tools

| Keybinding | Action |
| :--- | :--- |
| `Print Screen` | Select an area to screenshot (copies to clipboard) |
| `SUPER` + `SHIFT` + `C` | Open color picker (`hyprpicker`) |

## 🖱️ Mouse & Trackpad Actions

### Mouse Bindings
| Input | Action |
| :--- | :--- |
| `SUPER` + `LMB` (Hold & Drag) | Move floating window |
| `SUPER` + `RMB` (Hold & Drag) | Resize window |

### Gestures
| Input | Action |
| :--- | :--- |
| `3-Finger Swipe Horizontal` | Switch to next/previous workspace |
| `ALT` + `3-Finger Swipe Down` | Close active window |
| `SUPER` + `3-Finger Swipe Up` | Toggle fullscreen |
| `4-Finger Swipe Left` | Toggle floating mode |

## 🎵 Media & System Controls

| Hardware Key | Action |
| :--- | :--- |
| `Volume Up / Down` | Adjust system volume by 5% |
| `Audio Mute` | Toggle speaker mute |
| `Mic Mute` | Toggle microphone mute |
| `Brightness Up / Down`| Adjust screen brightness by 5% |
| `Media Play / Pause` | Play or pause current media |
| `Media Next / Prev` | Skip to next or previous track |
